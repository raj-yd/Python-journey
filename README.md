## ✅ What I've Learned So Far

### 🔢 Variables & Data Types
- Declaring variables (`name = "Alice"`, `age = 25`)
- Data types: `str`, `int`, `float`, `bool`
- Type conversion: `int()`, `str()`, `float()`
- F-strings for formatting: `f"Hello, {name}!"`

### 🔁 Loops
- `for` loops — iterating over lists, ranges, strings
- `while` loops — running until a condition is met
- `break` and `continue` — controlling loop flow
- Nested loops

### ⚙️ Functions
- Defining functions with `def`
- Parameters and return values
- Default arguments
- Calling functions and reusing code

## 💡 Tips I've Picked Up

- **Indentation matters** — Python uses whitespace to define blocks (no `{}` like other languages)
- **Read error messages carefully** — they usually tell you exactly what went wrong
- **Practice daily** — even 15 minutes of coding keeps the momentum going
- **Use the REPL** — `python3` in the terminal is great for quick experiments
